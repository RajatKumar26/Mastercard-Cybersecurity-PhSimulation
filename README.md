Mastercard Cybersecurity - Phishing Awareness Tool

This repository contains an automated solution and training documentation developed as part of the Mastercard Cybersecurity Job Simulation" hosted on Forage.

📌 Project Overview
The objective of this project is to simulate a security awareness scenario. It automates the generation of sophisticated phishing emails to help train Mastercard corporate employees against social engineering threats.

🛠️ Requirements Fulfilled
Contextual Reinforcement: Adds an urgent and realistic corporate security context.
Link Masking: Obfuscates simulation links within legitimate anchor text.
Grammar & Professionalism: Eliminates obvious indicators of a fake email to mimic high-level spear-phishing threats.

📋 Security Awareness Training Structure
1. Understanding Phishing: Identifying deceptive social engineering targeting corporate credentials.
2. Red Flag Detection: Analyzing artificial urgency, link masking, and look-alike domains.
3. Employee Defense: Implementation of verification guidelines and instant reporting protocols.

⚖️ Disclaimer
This project is for educational and security awareness simulation purposes only under the official Forage framework. It does not contain or interact with any real Mastercard corporate data or infrastructure.
